## 🎮 Геймплей

**Управление:**
- `WASD` / `Стрелки` — движение
- `E` — взаимодействие с объектами
- `ESC` — пауза / меню
- `M` — карта мира
- Мобильный: виртуальный D-Pad на экране

**Цель:**
- Собери 5 артефактов
- Активируй Маяк CRADLE
- Исследуй все 7 локаций (разблокируются по сюжету)
- Прокачай Battle Pass до 10 уровня
